# Factura-App
Generador de facturas dominicanas como Single Page Application (SPA) construida con React, Vite y Tailwind CSS. Permite registrar datos del cliente, agregar
productos/servicios, calcular totales con ITBIS y generar una factura lista para imprimir.

# Instalación y uso
## 1. Instalar dependencias
npm install

## 2. Iniciar el servidor de desarrollo
npm run dev

## 3. Abrir http://localhost:5173
